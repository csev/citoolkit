Using NT for a Web Server - A Case Study

This is an NT installation case study. This case study starts with a basic installation of NT, includes basic system administration tasks, installs and configures IIS 4.0, and installs FrontPage98.  Once the system is properly installed and configured, we backup the system, reformat the disk and perform a full recovery.

There are about 320 screen shots in this step by step guide to installing and maintaining NT.

You can click on the images to see larger versions of the screen-shots throughout the tutorial.  The images are rather large to include sufficient detail, so this is best viewed over a high speed connection or from a CD-ROM.

The tutorial is made up of fifteen separate exercises.  The exercises are designed to be performed in order.

  • Introduction to the Tutorial
  • Overview of the NT Administration Tutorial
  • Booting the system and interacting with BIOS
  • Installing NT from floppies
  • Setting up NT - Phase II
  • Testing your NT Installation
  • Post Installation configuration and administration
  • Installing Windows Option Pack and Internet Information Server V4.0
  • Installing Visual Interdev 6.0 Server Components
  • Patching Windows NT for Y2K compliance
  • Configuring and testing FTP using IIS 4.0
  • Configuring and testing WWW using IIS 4.0
  • Installing Visual Interdev 6.0 and FrontPage98 on a client system
  • Configuring and using FrontPage98
  • Performing Backups
  • Performing a full recovery after a system disaster
    • Installing a minimal NT - Step 1
    • Installing a minimal NT - Step 2
    • Recovering the NT data and operating system
