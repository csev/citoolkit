Community Information ToolKit
Hardware Requirements

Software

The toolkit project software will require/run on the following software:

  • Microsoft Windows NT 4.0
  • Microsoft Internet Information Server (IIS)
  • Microsoft Active Server Pages
  • (Optional) Microsoft Visual Interdev
  • (Optional) Microsoft FrontPage98

Minimum Server Hardware

The minimum hardware configuration required to support this software is as follows:

  • Pentium-II 300Mhz
  • 64MB RAM
  • 4GB Hard Disk
  • CD-ROM
  • 10/100 PCI-based Ethernet card

You will need to add an NT supported backup option to this hardware. We do not recommend anything below a Digital Audio Tape backup system connected using a SCSI controller. Other backup hardware (such as a Travan style tape or CD-ROM Writer) may work well enough for your purposes with some additional effort.

 

New Server Hardware

If a server is specifically being purchased for this project, the following hardware is recommended:

  • Pentium-II 300Mhz (Or higher)
  • 128MB RAM
  • 4GB Hard Disk (Either SCSI or Ultra DMA IDE)
  • 12GB Digital Audio Tape (DDS-3) Backup (SCSI Connection)
  • CD-ROM Reader (SCSI or IDE)
  • CD-ROM Writer (SCSI Connection) (optional)
  • 10/100 PCI-based Ethernet card

It is possible to order this type of system pre-configured with this equipment.

Completely configured systems meeting the minimum specification can be bought for under $5000. If your server is configured as a "server" class machine, it will usually include Ultra-SCSI disk and the option for arrays of disks, both of which improve performance for high-volume Web sites. Server class machines provide greater expandability and in some cases are designed to be less prone to failure. Costs of server class machines can range from $5000 to $10,000 or more.

Network

  • Permanent, direct connection to the Internet
  • Connection speed dependent on volume and complexity of information transferred
