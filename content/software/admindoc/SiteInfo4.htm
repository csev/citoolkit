Site Custom Field #1: A field that can be defined by the site -- used in the calendar to prompt for additional information about events.  (e.g., if "language" were used as a custom field, then events where spanish (or french)  is primarily spoken could be grouped.

Home Button URL (bottom of Calendar pages): Home URL for the organizations web presence.  This is only used if the organization has an existing web site. If this is set to "None" the Home button will not be shown. If this is not set (default), the the Home button will point to the main start page for the database showing the user the choices of calendar, forum, or administration.
