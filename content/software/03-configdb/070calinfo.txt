In this section, we set the calendar formatting and control information.

<b>Default Mode:</b> The default mode the event calendar will appear
for a user. Big Month, Month, List.

<b>Allow List, BigMonth, Month View </b>: You can use these to customize the types of views that your users can select.  You need to select at least one and include default view as one of those selected.

<b>Number of items per page</b>: The number of event items displayed
per page by default.

<b>Support Audience Type:</b> Yes/No. Enable support for
audience types.(e.g., children, families, seniors, etc.).

<b>Support Event Type:</b> Yes/No. Enable support for event
types. (e.g., meeting, concert, storytime, school, etc)

<b>Require Approval of Event:</b> Yes/No. Require approval of
new events.   If this is not checked, events will appear as soon as they are submitted. 

<b>Send Email to all approvers?:</b> Send an email message to ALL
approvers when an event is awaiting approval.
