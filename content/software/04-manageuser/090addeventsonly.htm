There are many roles which the user can perform. If you add them all, then this account is another administrator account.

The list of roles is as follows:

  • Add Events
  • Approve Events
  • Delete Events
  • Edit Events
  • Participate in a forum
  • Moderate Forums
  • Manage Forums
  • Administer the Site

Multiple roles can be selected using the shift or control keys.

In this example, we will only allow our user to Add Events.
